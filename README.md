📚 Smart Study Question Generator

A simple Streamlit web application that generates random study questions based on topic and difficulty level.
This tool helps students practice concepts quickly by generating random questions for revision.

🚀 Features

Generate random study questions

Choose topic (Python, SQL, AI)

Select difficulty level (Easy, Medium, Hard)

Choose the number of questions

Simple and interactive Streamlit interface

Great for quick practice and revision

🛠️ Technologies Used

Python

Streamlit

Random module

study-question-generator

▶️ How to Run the Project
1️⃣ Clone the repository
git clone https://github.com/your-username/smart-study-question-generator.git
2️⃣ Go to the project folder
cd smart-study-question-generator
3️⃣ Install dependencies
pip install streamlit
4️⃣ Run the application
streamlit run app.py
📖 How It Works

Enter a topic (Python / SQL / AI).

Select the difficulty level.

Choose the number of questions.

Click Generate Questions.

The app displays random questions for practice.

The app will start and open in your browser at:

http://localhost:8501/
